Способы навигации в Bootstrap

Выпадающее меню (Dropdown)

Компактная навигация, скрытая за кнопкой. Экономит место и идеально подходит для мобильных интерфейсов.

  • Классы: .dropdown, .dropdown-menu, .dropdown-item
  • Требует JavaScript для работы
  • Кнопка-переключатель с атрибутом data-bs-toggle="dropdown"

Вкладки-пилюли (Nav Pills)

Гибкий способ переключения контента без перезагрузки страницы. Активный пункт выделяется заливкой.

  • Классы: .nav, .nav-pills, .nav-link
  • Переключение через JavaScript (data-bs-toggle="pill")
  • Содержимое в .tab-content и .tab-pane

Навигационная панель (Navbar)

Главное меню сайта с логотипом и ссылками. Поддерживает адаптивное сворачивание в гамбургер.

  • Классы: .navbar, .navbar-brand, .navbar-nav
  • Адаптивность через .navbar-expand-*
  • Цветовые схемы: data-bs-theme="dark" или .navbar-dark
